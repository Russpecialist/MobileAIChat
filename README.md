MobileAIChat - Чат-приложение с ИИ
MobileAIChat - это мультиплатформенное приложение для общения с искусственным интеллектом, разработанное с использованием Flutter. Приложение поддерживает OpenRouter и VseGPT и позволяет работать с различными языковыми моделями через единый интерфейс.

Что происходит в авторизации
Первый вход: пользователь вводит API ключ.
Приложение автоматически определяет провайдера:
sk-or-v1-... -> OpenRouter
Выполняется проверка баланса по API.
Если ключ валидный и баланс положительный:
генерируется 4-значный PIN,
PIN и ключ сохраняются в локальной базе данных.
Следующие входы выполняются по PIN.
Есть сброс ключа и повторная регистрация.
Поддержка API
OpenRouter.ai
Баланс отображается в долларах ($)
Стоимость сообщений рассчитывается в долларах за миллион токенов
Широкий выбор языковых моделей от разных провайдеров
Единый API для доступа к различным моделям
VseGPT
Баланс отображается в рублях (₽)
Стоимость сообщений рассчитывается в рублях
Поддерживается автоматическое определение провайдера по ключу

Начало работы
Клонируйте репозиторий: git clone https://github.com/GarryFerstein/MobileAIChat.git
Перейдите в директорию проекта
cd DesktopAIChat
Создайте файл .env на основе .env.example
Установите зависимости: flutter pub get
Запустите приложение (flutter run -d ios для iOS Simulator или через Android Studio/Xcode)
Введите API ключ в самом приложении при первом запуске
Следуйте инструкциям в INSTALL.md для детального запуска и тестирования
Быстрый запуск на iOS Simulator
flutter pub get
cd ios && pod install && cd ..
open -a Simulator
flutter run -d ios
Конфигурация
Для работы приложения необходимо указать в файле .env:

OPENROUTER_API_KEY - можно оставить пустым (ключ вводится в UI при первом входе)
BASE_URL - можно оставить пустым (определяется автоматически по ключу)
MAX_TOKENS - Максимальное количество токенов для ответа
TEMPERATURE - Температура генерации (0.0 - 1.0)
Пример .env:

OPENROUTER_API_KEY=
BASE_URL=
MAX_TOKENS=1000
TEMPERATURE=0.7
